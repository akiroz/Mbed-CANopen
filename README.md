# Mbed-CANopen

CANopen library for mbedOS 6.

### Features:
- [x] NMT master
  - [x] Module Control Service
  - [x] Error Control Service
    - [ ] Guarding
    - [x] Heartbeat
- [ ] NMT slave
- [x] SDO client
  - [x] Upload
    - [x] Expedited
    - [ ] Segmented
  - [x] Download
    - [x] Expedited
    - [ ] Segmented
- [ ] SDO server
- [x] PDO producer
  - [x] Push
  - [ ] Pull
  - [ ] Sync
- [x] PDO consumer
  - [x] Push
  - [ ] Pull
- [ ] SYNC producer
- [ ] SYNC consumer
- [ ] EMCY producer
- [ ] EMCY consumer
